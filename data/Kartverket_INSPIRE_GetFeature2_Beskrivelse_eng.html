Get maximum 50 geographical names within a geographical rectangle and of type "fyrlykt" ( Lighthouse (Sector light) ).

GetFeature operation - get objects of type "NamedPlace":
  • Max number (count): 50
  • Object types (typeNames): elf-gn:NamedPlace
  • Attribute Projection (PropertyName): none
  • Selection (Filter): A geographical area and a given name type specified by fes:And, fes:BBOX og fes:PropertyIsEqualTo
  • Sorting (SortBy): none
Geographical names are widely used in every-day communication for referring to various natural and man-made objects in the real world. The main value of the INSPIRE geographical names model is that geographical names can be used as a spatial property in another theme of INSPIRE. http://European Location Framework (ELF)
