Get Plan outlines on Mulicipality plans for a given Municipality.

GetFeature-operation - get objects of type "KpOmråde" (Mulicipality plans):
  • Object types (typeNames): app:KpOmråde
  • Attribute Projection (PropertyName): none
  • Selection (Filter): Objects with a given Municipality ID specified by fes:PropertyIsEqualTo
  • Sorting (SortBy): none
The Plan outlines represents geographic data with associated properties and will include a number of areas with different considerations and often several vertical levels. http://www.kartverket.no/en/
