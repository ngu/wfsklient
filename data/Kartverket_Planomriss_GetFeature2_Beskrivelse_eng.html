Get all Plan outlines on Zoning for a given municipality, which came into effect after a given date and sort it on that date.

GetFeature-operation - get objects of type "RpOmråde" (Zoning):
  • Object types (typeNames): app:RpOmråde
  • Attribute Projection (PropertyName): none
  • Selection (Filter): Objects with a given mulicipality ID and effective date after the given date specified by fes:And, fes:PropertyIsEqualTo and fes:PropertyIsGreaterThanOrEqualTo
  • Sorting (SortBy): Sort by "app:ikrafttredelsesdato" (effective date)
The Plan outlines represents geographic data with associated properties and will include a number of areas with different considerations and often several vertical levels. http://www.kartverket.no/en/
