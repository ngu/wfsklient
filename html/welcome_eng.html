Guideline and Web Client for WFS

A "Norway digital" project which prepares a technical WFS guideline with examples and promotes the use of WFS through a reusable Web Client:

  • Create guidelines on how to get started with WFS for Norway digital partners.
  • Addressing the demands for use of the WFS.
  • Try to limit the different implementation methods.
  • Show practical examples of configuration and queries.
  • Give examples using both WMS and WFS from the same service.
  • The WFS group represents a broad technical background, both from the private and public sectors.
  • Agreement within the group is important to ensure academic breadth and ease of use in practice.
  • The guideline must also be coordinated with similar projects like Geosyncronizing and GML.
  • The WFS Client is intended as a supplement to the guide.

The application has a simple MVC-based code that is reusable for other contexts. Running the predefined queries, editing and uploading including validation of custom queries. Reuse of explanations and figures from the guideline in separate description tabs.
